Motherboard adalah papan sirkuit utama pada komputer yang berfungsi sebagai penghubung dan pengatur kerja semua komponen.
berikut cara kerja nya
1. Mendistribusikan daya
Listrik dari power supply masuk ke motherboard
Motherboard menyalurkan daya ke prosesor, RAM, chipset, dan komponen lain
Tegangan diatur agar sesuai dengan kebutuhan tiap komponen
2. Menghubungkan semua komponen
Motherboard menjadi jalur komunikasi antara:
Prosesor
RAM
Penyimpanan (HDD / SSD)
Kartu grafis
Perangkat input dan output
Semua data lewat jalur sirkuit (bus) di motherboard.
3. Mengatur komunikasi data
Chipset mengatur aliran data antara prosesor, RAM, dan perangkat lain
Jalur seperti PCIe, SATA, dan USB digunakan untuk pengiriman data
Kecepatan tergantung jenis motherboard
4. Proses booting komputer
Saat komputer dinyalakan:
BIOS/UEFI melakukan pengecekan perangkat (POST)
Memastikan CPU, RAM, dan perangkat lain berfungsi
Jika normal, sistem operasi dijalankan dari penyimpanan ke RAM
5. Mengatur kerja perangkat tambahan
Motherboard mengontrol:
Port USB
Audio
LAN
Kipas dan sensor suhu
Kesimpulan
Motherboard bekerja sebagai pusat penghubung dan pengatur, menyalurkan daya, mengatur komunikasi data, dan memastikan semua komponen komputer dapat bekerja bersama dengan baik.
