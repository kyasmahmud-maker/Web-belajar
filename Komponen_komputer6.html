VGA atau GPU adalah komponen yang bertugas mengolah data grafis agar dapat ditampilkan ke layar monitor.
berikut cara kerja nya
1. Data grafis dikirim ke GPU
Prosesor mengirim perintah grafis ke GPU
Perintah ini berisi informasi gambar, bentuk, warna, dan posisi
---
2. Pemrosesan grafis oleh GPU
GPU memproses data secara paralel, meliputi:
Perhitungan bentuk (vertex processing)
Pengolahan tekstur dan warna
Perhitungan cahaya dan bayangan
Rendering gambar menjadi frame
GPU memiliki banyak core kecil untuk mengolah data grafis dengan cepat.
---
3. Penyimpanan sementara di VRAM
Hasil perhitungan grafis disimpan di VRAM (Video RAM)
VRAM menyimpan tekstur, frame, dan data visual
VRAM lebih cepat dibanding RAM biasa
---
4. Pengiriman ke monitor
Gambar yang sudah selesai dirender dikirim ke monitor
Melalui port HDMI, DisplayPort, atau VGA
Proses ini terjadi puluhan hingga ratusan kali per detik
---
5. Sinkronisasi tampilan
GPU menyesuaikan jumlah frame dengan refresh rate monitor
Bertujuan agar tampilan halus dan tidak patah-patah
---
Perbedaan GPU terintegrasi dan diskrit
GPU terintegrasi: menyatu dengan prosesor, menggunakan RAM sistem
GPU diskrit: kartu terpisah, memiliki VRAM sendiri, performa lebih tinggi
---
Kesimpulan
GPU bekerja dengan menerima perintah grafis dari prosesor, memprosesnya secara paralel, menyimpan hasil di VRAM, lalu menampilkan gambar ke monitor dengan cepat dan halus.
