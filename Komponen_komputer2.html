semangat belajar

RAM adalah memori sementara yang digunakan komputer untuk menyimpan data dan program yang sedang dipakai oleh prosesor.
berikut cara kerja ram
1. Data dimuat ke RAM
Saat kamu membuka aplikasi atau file:
Data dari penyimpanan (HDD / SSD) dipindahkan ke RAM
Karena RAM jauh lebih cepat dari penyimpanan, prosesor bisa bekerja lebih cepat
2. Prosesor mengakses RAM
Prosesor mengambil data dan instruksi langsung dari RAM
RAM bisa diakses secara acak (random), tidak perlu urut
Ini membuat proses berjalan cepat
---
3. RAM menyimpan data sementara
Data aplikasi yang sedang berjalan disimpan di RAM
Jika banyak aplikasi dibuka, penggunaan RAM akan meningkat
Jika RAM penuh, sistem akan melambat
4. Data dihapus saat listrik mati
RAM bersifat volatile (data hilang saat komputer mati)
Berbeda dengan HDD/SSD yang menyimpan data secara permanen
Hubungan RAM dengan prosesor
RAM = tempat kerja sementara
Prosesor = pekerja
HDD/SSD = gudang penyimpanan
Tanpa RAM, prosesor harus mengambil data langsung dari penyimpanan dan itu sangat lambat.
Kesimpulan
RAM bekerja dengan menyimpan data sementara yang sedang digunakan agar prosesor dapat mengaksesnya dengan cepat. Semakin besar dan cepat RAM, semakin lancar komputer menjalankan banyak aplikasi.
