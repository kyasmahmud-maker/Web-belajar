Storage adalah komponen yang berfungsi menyimpan data secara permanen, baik saat komputer hidup maupun mati.
berikut cara kerja nya
1. Penyimpanan data
Data seperti sistem operasi, aplikasi, foto, dan video disimpan di storage
Data tetap ada walaupun komputer dimatikan
2. Saat komputer dinyalakan (booting)
Motherboard memerintahkan storage mengirim data sistem operasi
Data dikirim ke RAM
Prosesor mulai menjalankan sistem operasi
3. Akses data oleh prosesor
Prosesor tidak langsung membaca storage
Data dipindahkan dulu ke RAM
RAM mengirim data ke prosesor untuk diproses
4. Penulisan dan pembacaan data
Saat menyimpan file, data ditulis ke storage
Saat membuka file, data dibaca dari storage
Kecepatan tergantung jenis storage
Jenis storage dan cara kerjanya
Hard Disk Drive (HDD)
Menggunakan piringan berputar
Kepala baca tulis mengambil data secara magnetik
Lebih lambat, kapasitas besar
Solid State Drive (SSD)
Menggunakan chip memori
Tidak ada bagian bergerak
Akses data sangat cepat
NVMe SSD
Menggunakan jalur PCIe
Kecepatan lebih tinggi dari SSD SATA
Cocok untuk performa tinggi
Kesimpulan
Storage bekerja dengan menyimpan data secara permanen, mengirim data ke RAM saat dibutuhkan, dan menerima data baru saat penyimpanan. Semakin cepat storage, semakin cepat komputer booting dan membuka aplikasi.
